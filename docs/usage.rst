=====
Usage
=====

To use labelx in a project::

    import labelx

To use the command line functionality and see the package information -

.. code-block:: shell

   labelx pkg-info

This should output something similar as below -

.. code-block:: shell

   -------------------------------- Package Information --------------------------------
   [*] Package Name: labelx
   [*] Version: 1.0.1
   [*] License: GNU General Public License v3
   -------------------------------------- Goodbye! -------------------------------------
